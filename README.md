# PsinCv1

`Project #1 for PSIN`

## Dependencies

- Java 8+

## Installation

You can install this project through Maven's Central Repository. In your
`pom.xml`:

```
<dependencies>
  <dependency>
    <groupId>sk.upjs.paz</groupId>
    <artfiactId>PsinCv1</artifactId>
    <version>1.0-SNAPSHOT</version>
  </dependency>
</dependencies>
```
